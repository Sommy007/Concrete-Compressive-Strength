# Concrete-Compressive-Strength

PROJECT OVERVIEW
Concrete is a critical material in the construction industry, and its compressive strength is one of the most important indicators of structural quality and safety. Determining concrete strength through traditional laboratory testing can be time-consuming, as samples must cure for a specific period before testing. This can delay decision-making and increase project costs.

The compressive strength of concrete is influenced by several factors, including the proportions of cement, water, fly ash, blast furnace slag, aggregates, superplasticizer, and the age of the concrete. Because the relationship between these variables and concrete strength is highly nonlinear, accurately predicting strength can be challenging using conventional methods.

This project leverages machine learning techniques to predict concrete compressive strength based on material composition and curing age. By developing and evaluating regression models on the UCI Concrete Compressive Strength dataset, the project demonstrates how data-driven approaches can support faster strength estimation, improve quality control, and assist engineers in optimizing concrete mix designs for construction applications.


PROJECT GOAL 
This project is aimed at using supervised machine learning (regression) to predict the compressive strength of high-performance concrete.


VARIABLES
Target (Output):
Concrete compressive strength (MPa)

Features (Input):
Cement (kg/m³), Blast Furnace Slag (kg/m³), Water (kg/m³), Superplasticizer (kg/m³), Fly Ash (kg/m³), Coarse Aggregate (kg/m³), Fine Aggregate (kg/m³) and Age (day)
